📡 1. 系统总体设计思路

本系统模拟“微型防空反导体系”，通过视觉感知 + AI目标识别 + 高速云台伺服 + 脉冲激光烧蚀，实现室内静止或飞行蚊子的全自动灭杀。工作流程如下：

🔁 闭环流程： 广角/长焦摄像头采集图像 → 传入Jetson Nano运行轻量化目标检测模型(识别蚊子) → 获取蚊子的像素坐标 → 利用PID算法解算云台角度偏差 → 驱动数字舵机对准 → 当瞄准误差小于阈值(5~10像素) → 触发激光脉冲(10~50ms) → 击落/碳化蚊子 → 下一目标。

✅ 特色安全设计： 红外热释电模块监测人宠活动，一旦进入警戒区域，硬件级切断激光器供电，确保绝对安全。同时软件逻辑仅在室内光线较暗时激活避免干扰。

🎯 探测与识别

采用USB摄像头(可搭配被动红外补光)，运行经剪枝训练的YOLOv8n或SSD-MobileNet模型。针对蚊子飞行姿态、轮廓进行数据增强，实现实时推理速度 >20FPS。当蚊子悬停或低速移动时锁定最佳。

⚙️ 伺服与打击

2轴高精度金属舵机云台，配合PCA9685驱动，实现0.2°级指向。5W 445nm蓝光激光模组配合脉冲驱动板，瞬间高温气化蚊虫。通过散热风扇保证长期运行。

🔧 4. 完整硬件组装 & 软件开发流程

🛠️ 4.1 机械与电气组装

  • 步骤1： 将激光模组与摄像头平行固定于云台上平面，使用铝型材或3D打印支架，保证光轴与摄像头视野中心对齐（激光校准：10米外照射靶点与画面中心十字重合）。
  • 步骤2： 将两个数字舵机安装在云台底座，并固定在底板，确保转动无干涉。连接舵机线至PCA9685驱动板。
  • 步骤3： 激光驱动板的 TTL 信号连接主控GPIO（PWM引脚），激光电源串接安全继电器，继电器控制端接主控IO。红外热释电传感器输出接主控外部中断引脚，同时硬件上串联继电器使能端（双重保险）。
  • 步骤4： 采用12V锂电池组为舵机、激光器主供电，5V降压模块为主控、摄像头、传感器供电。添加总开关和急停按键。
  • 步骤5： 通电前测试舵机回中，并确保激光器仅在校准模式下低压点亮。加装物理挡光板限制射界外溢。

💻 4.2 软件算法与部署

  • 环境配置： Jetson Nano 烧录JetPack 4.6+，安装CUDA、cuDNN、OpenCV、TensorRT及Ultralytics YOLO。
  • 数据集获取与训练： 收集蚊子图像(飞行+静止)约1500张，标注bounding box。利用预训练模型微调，导出engine推理引擎。
  • 主控逻辑实现： Python脚本循环抓帧→推理获取目标中心(cx,cy)→调用PID控制器→输出PWM调整舵机→等待系统稳定→误差小于阈值时，激光点火(发射脉冲10~30ms)→记录击杀。增加目标滤波、防抖。
  • 安全防护代码： 持续监测热释电传感器及全局标志位，如果检测到人或宠物活动，立即释放继电器断开激光电源，并停止追踪动作，直到安全区域恢复。
  • 优化实战： 针对蚊子移动速度较快，可采用卡尔曼滤波预测下一帧位置，提升命中率。设置最大激光发射间隔限制防止过热。
💡 调试经验： 激光瞄准校准是最大难点，建议先在静态靶板(黑色蚊子剪纸)上调校；飞行蚊子实验必须在密闭、无反射、无人区域测试。激光功率从1W起逐步增加。

📌 6. 完整调试指引 & 性能预期

✅ 预期灭蚊效率： 室内静态或悬停蚊子击杀率约75%-90%（光照充足且距离<2米），飞行中击落率约60%，随着PID优化和更佳AI模型可提升。激光精准命中可瞬间击落并产生轻微焦糊味。

✅ 系统限制： 视野范围受限于摄像头FOV及云台旋转角度(水平0~270°，俯仰±45°)，适合书房/卧室角落部署。工作噪音主要为风扇与舵机运动，约45分贝。

✅ 重要安全提醒： 激光不可直视，本设备严禁在开放空间未加物理防护罩运行，必须保证无反射镜面，同时使用传感器自动停发。请遵守GB 7247.1激光安全标准。
